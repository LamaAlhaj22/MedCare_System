Project Description:
MedCare is a distributed medical appointment booking system that allows patients to book appointments, doctors to manage their schedules, and administrators to control and monitor the system.
The system is designed to follow distributed system principles and includes features such as authentication, CRUD operations, and database integration.

Main Features:
User Registration and Login

Book, Edit, and Cancel Appointments
Doctor Dashboard for managing appointments
Admin Panel for managing doctors and system data

System Roles:
Patient → Books and manages appointments
Doctor → Views and updates appointment status
Admin → Manages doctors and monitors the syste
